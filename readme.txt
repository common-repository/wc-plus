== Description ==

A WooCommerce Checkout plugin, providing a conversion-optimized, responsive template. Designed & built by a team with the experience and delivery of 300+ WooCommerce websites.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/wc-plus-free-version' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->WC Plus screen to configure the plugin.

== Frequently Asked Questions ==

= How do I install the plugin? =

You can install the plugin by uploading the plugin files to the `/wp-content/plugins/wc-plus-free-version` directory, or by installing the plugin through the WordPress plugins screen directly.

= Is this plugin compatible with the latest version of WooCommerce? =

Yes, this plugin is tested up to WooCommerce version 8.9.1.
